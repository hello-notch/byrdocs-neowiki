## 一、单项选择题（20 分，每空 2 分）

1. 已知 8421BCD 码 $01011001$，其对应的十进制数为 <Slot />。

<Choices>
    + 59
    - 75
    - 110
    - 89
</Choices>

<Solution>
解析：8421BCD 码中，每四位二进制数表示一位十进制数。$0101$ 对应 5，$1001$ 对应 9，因此十进制数为 59。
</Solution>

2. 一个班级有 4 个班委委员，如果要开班委会，这四个委员全部同意才可以召开，其逻辑为 <Slot /> 逻辑。
<Choices>
    - 或
    - 非
    + 与
    - 或非
</Choices>

<Solution>
解析：“全部同意才可以召开”体现的是“只有所有条件都满足，结果才成立”的逻辑关系，即与（AND）逻辑。
</Solution>

3. 在一个三变量函数中，以下 <Slot /> 为最小项。

<Choices>
    - $A$
    - $ACA$
    - $ABB$
    + $ABC$
</Choices>

<Solution>
解析：最小项（Minterm）要求包含逻辑函数的所有变量，且每个变量以原变量或反变量的形式出现且仅出现一次。只有 D 选项 $ABC$ 包含了全部三个变量。
</Solution>

5. 同或门的输入分别为 1 和 0，其输出为 <Slot />。

<Choices>
    + 0
    - 1
    - X
    - 无法判断
</Choices>

<Solution>
解析：同或门（XNOR）的逻辑是“相同出 1，相异出 0”。输入 1 和 0 不相同，故输出为 0。
</Solution>

6. 使逻辑函数 $F=(A'+B')(B+C')(A'+C)$ 为 0 的逻辑变量组合为 <Slot />。

<Choices>
    - $ABC=000$
    - $ABC=010$
    - $ABC=011$
    + $ABC=110$
</Choices>

7. 逻辑式 $AB+AC+C$ 与下列 <Slot /> 相同。

<Choices>
    - $AB$
    - $AC+B$
    - $A$
    + $AB+C$
</Choices>

<Solution>
解析：化简过程为 $AB+AC+C=AB+C(A+1)$。因为 $A+1=1$，所以原式 $=AB+C$。
</Solution>

8. 在逻辑式 $F=AB+CD$ 中，输入 $A=1$，$B=0$，$C=1$，$D=1$，则 $F$ 为 <Slot />。

<Choices>
    - 0
    + 1
    - X
    - 无法判断
</Choices>

<Solution>
解析：将输入代入表达式，$F=AB+CD=(1\times0)+(1\times1)=0+1=1$。
</Solution>

9. 在四变量卡诺图中，逻辑上不相邻的一组最小项为 <Slot />。

<Choices>
    - $m_1$ 与 $m_3$
    - $m_4$ 与 $m_6$
    - $m_5$ 与 $m_{13}$
    + $m_2$ 与 $m_8$
</Choices>

<Solution>
解析：卡诺图中逻辑相邻意味着两个最小项的二进制代码只有一位不同。$m_2(0010)$ 和 $m_8(1000)$ 有两位不同（最高位和倒数第二位），因此不相邻。
</Solution>

10. 下列几种说法中错误的是 <Slot />。

<Choices>
    - 任何逻辑函数都可以用卡诺图表示。
    - 逻辑函数的卡诺图是唯一的。
    - 同一个卡诺图化简结果可能不是唯一的。
    + 卡诺图中 1 的个数和 0 的个数是相同的。
</Choices>

<Solution>
解析：卡诺图是真值表的几何表示，图中 1 的个数和 0 的个数完全取决于该逻辑函数本身的输出状态，不一定相同。
</Solution>

## 二、填空题（10 分）

1. $(11.001)_2=($<Blank>3.2</Blank>$)_{16}=($<Blank>3.125</Blank>$)_{10}$

<Solution>
解析：二进制转十六进制：整数部分不足四位补零（$0011=3$），小数部分不足四位补零（$0010=2$），得3.2。

二进制转十进制：$1\times2^1+1\times2^0+1\times2^{-3}=2+1+0.125=3.125$。
</Solution>

2. $(8F.FF)_{16}=($<Blank>10001111.11111111</Blank>$)_2=($<Blank>143.99609375</Blank>$)_{10}$

<Solution>
解析：十六进制转二进制：$8=1000$，$F=1111$，直接转换即可。

十六进制转十进制：整数部分 $8\times16+15=143$；小数部分 $15/16+15/256=255/256=0.99609375$。
</Solution>

3. $(10101101)_{\text{原码}}=($<Blank>11010010</Blank>$)_{\text{反码}}=($<Blank>11010011</Blank>$)_{\text{补码}}$

4. 函数 $F=\overline{A}+\overline{\overline{AB}+A(C+D)}$，其反函数 $F'=$<Blank>$A\left((AB)'+A(C+D)\right)$</Blank>；对偶式$\overline{F}=$<Blank>$A'\left((A+B)'(A+CD)\right)'$</Blank>。
